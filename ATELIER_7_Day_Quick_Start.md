## DAY 2: LANDING PAGE DEPLOYMENT (1.5 hours)

### Task 1: Set Up Vercel (30 min)
1. Go to vercel.com
2. Sign up with GitHub account (or create account)
3. Install Vercel GitHub integration

### Task 2: Deploy Landing Page (30 min)
1. Create GitHub repo: "atelier" (or "atelier-website")
2. Add the HTML from **ATELIER_Landing_Page.md** file
3. Create file: `index.html`
4. Push to GitHub
5. Vercel auto-deploys
6. You get: `atelier.vercel.app`

### Task 3: Connect Custom Domain (30 min)
1. Go to Vercel dashboard
2. Select your project
3. Go to "Domains" tab
4. Add your domain (atelierdance.com)
5. Follow instructions to point domain to Vercel

**Your landing page is live in 1.5 hours.**

### Task 4: Set Up Email Capture (30 min)
1. Go to mailchimp.com
2. Sign up (free account)
3. Create audience "Atelier"
4. Create signup form
5. Get embed code
6. Add to landing page (replace email signup code)

OR use Zapier:
1. Connect form → Google Sheet
2. Every email signup goes to your tracking sheet
3. Easy to manage

**Test:** Fill out the form yourself to make sure it works

## DAY 4: INTERVIEW RECRUITMENT (2-3 hours)

### Task 1: Make Recruitment List (1 hour)
Create Google Sheet with:
- Column A: Name
- Column B: Type (Pro / Amateur)
- Column C: Dance Style
- Column D: Contact (Instagram / Email / Phone)
- Column E: Status (To Contact / Contacted / Scheduled / Interviewed)

**List people to interview:**
- 15-20 professional dancers (choreographers, teachers, performers)
- 20-25 amateur dancers (serious hobby dancers)

**Where to find them:**
- Local dance studios (visit in person)
- Instagram hashtags (#[city]dancers, #[city]choreographer)
- Facebook dance groups
- Dance company websites

### Task 2: Write Recruitment Scripts (30 min)

**In-person approach (at dance studios):**
```
Hi! I'm doing research on the dance community in [city]. 

I'm building something to help dancers connect and collaborate, and I'd love to get your perspective. 

Would you have 10 minutes for a quick chat? Coffee's on me.
```

**Instagram DM approach:**
```
Hey [Name]! 

Love your work / dancing style. I'm researching the dance community in [city] and building something to solve the biggest pain points.

Would you be open to a quick 10-min call this week about how you currently find collaborators and book spaces?

Thanks,
[Your name]
```

**Facebook group post:**
```
Hi all! I'm researching the dance community in [city].

Quick 10-min research call about:
- How you find collaborators
- Where you practice/rehearse
- What you wish existed

Interested? Comment below or DM me.

(Building something to solve this 👇)
```

### Task 3: Send First Batch of Recruitment Messages (1.5 hours)
1. Visit 2-3 dance studios in person
   - Chat with dancers/teachers
   - Get email/phone for 5-10 people
2. Send 15-20 DMs on Instagram (use script above)
3. Post in 3-5 Facebook dance groups

**Target:** Get 5-10 coffee chats or Zoom calls scheduled by end of week

## DAY 5: SOCIAL MEDIA LAUNCH (1.5 hours)

### Task 1: Create First Instagram Post (45 min)
**Content:** Carousel about dancer struggles

**Image 1:** Bold title "What Dancers Are Asking For"

**Images 2-5:** Quotes (or if you don't have interviews yet, generic pain points):
```

"Studio rental costs more than my groceries"
"I want to teach but have no way to reach students"
```

```
• No central place to find collaborators
• Studio rental is expensive
• Teaching opportunities scattered

We're building one.

It's called Atelier.

#[City]Dancers #DanceLife #DanceCommunity #Atelier
```

**Post:** Schedule for Monday 6pm (use Buffer)

## FINAL REMINDERS

1. **You don't need perfect.** You need real. Imperfect > polished.
2. **Talk to dancers more than you code.** First 3 months = 70% listening, 30% building.
3. **Consistency beats virality.** 3 posts/week for 12 weeks > 10 posts once.
4. **Your network is your asset.** Who do you know who dances? Start there.
5. **Trust the process.** In 12 weeks, you'll look back amazed at progress.
