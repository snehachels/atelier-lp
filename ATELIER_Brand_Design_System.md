### Color Psychology for Atelier

- **Rust (#B85C38):** Warmth, creativity, artistic energy. Signals "this is a creative space."
- **Sage (#8B9D83):** Growth, community, balance. Represents the collaborative nature of dance.
- **Warm Dark (#4A4A42):** Professional, grounded, serious about craft. Warm undertone feels artistic, not corporate.
- **Cream (#F4EDE4):** Welcoming, studio-like, spacious. The feeling of walking into an art studio.
- **Medium Gray (#7F8178):** Readable, sophisticated body text. Complements the warm palette without harshness.

- **Colorblind-friendly:** Palette works for deuteranopia (red-green colorblindness) — validated
- **No color-only information:** Always pair colors with icons, text, or patterns

## 2. TYPOGRAPHY

### Font Pairing: Montserrat + Crimson Text (Recommended)

#### Montserrat
- **Type:** Geometric sans-serif
- **Best for:** Headlines, UI, buttons, branding
- **Weight options:** 500 (medium), 600 (semibold), 700 (bold)
- **Why:** Modern but timeless. Clean and elegant without feeling corporate.
- **Google Fonts:** `https://fonts.googleapis.com/css2?family=Montserrat:wght@500;600;700`

#### Crimson Text
- **Type:** Editorial serif
- **Best for:** Body copy, storytelling, long-form content
- **Weight options:** 400 (regular), 600 (semibold)
- **Why:** Classical, warm, readable. Evokes book design and editorial credibility.
- **Google Fonts:** `https://fonts.googleapis.com/css2?family=Crimson+Text:wght@400;600`

### Type Hierarchy

```
H1 (Headlines)
Font: Montserrat Bold (700)
Size: 56-72px (desktop) | 36-48px (mobile)
Line height: 1.1
Letter spacing: -1px
Color: #4A4A42 (Warm Dark) or #F4EDE4 (Cream on dark bg)
Use on: Landing page hero, major section titles

H2 (Subheadings)
Font: Montserrat Semibold (600)
Size: 28-36px
Line height: 1.2
Letter spacing: 0px
Color: #4A4A42 (Warm Dark)
Use on: Section headers, "Why Atelier", feature titles

H3 (Tertiary Headers)
Font: Montserrat Semibold (600)
Size: 20-24px
Line height: 1.3
Color: #4A4A42 (Warm Dark)
Use on: Card titles, feature names

Body Text
Font: Crimson Text Regular (400)
Size: 16-18px (desktop) | 16px (mobile)
Line height: 1.7-1.8
Letter spacing: 0px
Color: #7F8178 (Medium Gray)
Use on: Paragraph copy, descriptions

Small Text (labels, captions)
Font: Montserrat Medium (500)
Size: 12-13px
Line height: 1.4
Letter spacing: 0.5px
Color: #7F8178 (Medium Gray)
Use on: Labels, button text, helper text

Button Text
Font: Montserrat Semibold (600)
Size: 13-14px
Line height: 1.2
Letter spacing: 0.5px
Text transform: UPPERCASE
Color: White or #F4EDE4 (Cream)
Use on: CTA buttons, form buttons
```

### Typography Principles for Atelier

1. **Serif for story, sans-serif for action** — Use Crimson Text when telling the Atelier story; use Montserrat for CTAs and UI.
2. **Generous line height** — Make copy feel open and studio-like (1.7-1.8 for body).
3. **Moderate letter spacing** — Especially on all-caps button text (0.5px) for sophistication.
4. **Size contrast matters** — H1 should be significantly larger than H2 to create visual hierarchy.

## 3. LOGO & VISUAL MARK

### Recommended: Wordmark (Type-First Branding)

**Logo Structure:**
- Primary wordmark: "ATELIER" in Montserrat Bold, letter-spacing 1-2px
- Optional accent: Horizontal line beneath (Rust #B85C38, 2-3px thickness)
- Color options: Charcoal on Cream (primary) | Cream on Charcoal (reverse)

**Logo Sizes & Clearance:**
- Minimum size: 80px (width)
- Minimum clear space around: 10px on all sides
- Never stretch, rotate, or modify the letterforms

**Logo Variations:**
```
Version 1: "ATELIER" wordmark only (primary)
Version 2: "ATELIER" + horizontal line accent (secondary)
Version 3: "A" monogram mark (for favicons, app icons)
Version 4: "ATELIER" stacked vertically (social profiles)
```

**Why Wordmark?**
- Timeless and editorial (like fashion/art brands)
- Works at any size (especially small digital spaces)
- Sophisticated without requiring illustration skills
- Easy to scale to any context

### Alternative: Minimalist "A" Mark

If you want a symbol mark:
- Simple geometric "A" in Rust (#B85C38)
- Can stand alone or pair with "ATELIER" wordmark
- Works for app icons, favicon, profile pictures
- Inspired by movement/dance geometry (clean lines)

## 4. LAYOUT & SPACING

### Whitespace Philosophy

Atelier should feel like a **creative studio — spacious, breathing, intentional**.

**Spacing scale:**
- 8px (micro spacing, e.g., between icon + text)
- 16px (small, e.g., padding within cards)
- 24px (medium, e.g., section padding)
- 40px (large, e.g., between sections)
- 60px (extra large, e.g., between major sections)

**Container widths:**
- Mobile: Full width with 20px padding on sides
- Tablet: 768px content width, centered
- Desktop: 1100px content width, centered
- Max width never exceeds 1200px

### Grid System

Use a **12-column grid** with 16px gutters:
- 2-column layouts collapse to 1-column on mobile
- 3-column feature cards → 2 on tablet → 1 on mobile
- Never cramped; generous whitespace is part of the brand

## 5. DESIGN ELEMENTS

### Buttons

**Primary CTA Button:**
- Background: Rust (#B85C38)
- Text: White or Cream (#F4EDE4)
- Padding: 16px horizontal × 48px vertical
- Border-radius: 2px (nearly square, modern)
- Font: Montserrat Semibold (600), 13px, UPPERCASE
- Hover state: Background → Deep Burgundy (#640A14)
- Transition: 0.3s ease

**Secondary Button:**
- Background: Transparent
- Border: 2px Rust (#B85C38)
- Text: Rust (#B85C38)
- Hover: Background → Cream (#F4EDE4)

### Cards & Containers

- **Card background:** White or Cream (#F4EDE4)
- **Card border:** 1px Light Gray (#DDD) or left accent border (4px Rust #B85C38)
- **Shadow:** Subtle (0 2px 8px rgba(0,0,0,0.08))
- **Padding:** 40-50px
- **Border radius:** 2px (minimal, sophisticated)

### Dividers & Lines

- **Section dividers:** 1px Warm Tan (#D4B9A0)
- **Accent lines:** 2-3px Rust (#B85C38) or Sage (#8B9D83)
- **Never use pure black** — always use Charcoal (#3A3A3A) or Medium Gray (#7F8178)

### Icons

- **Style:** Line-based, minimal, 2px stroke weight
- **Color:** Rust (#B85C38) for primary actions, Charcoal (#3A3A3A) for secondary
- **Size:** 24px (small), 32px (medium), 48px (large)
- **No filled icons** — use outline/stroke only

## 6. IMAGERY & PHOTOGRAPHY

### Photography Style

- **Real dancers**, not stock photos or illustrated figures
- **Candid, in-motion moments** — rehearsals, not posed shoots
- **Studio lighting** — warm, intimate, not bright/outdoorsy
- **Diverse bodies, styles, ethnicities** — all dancers welcome
- **Context visible** — show the studio space, the community, the collaboration

### What to Photograph

- Dancers mid-rehearsal, focused and in motion
- Small groups collaborating, looking at choreography together
- Teachers adjusting a student's form
- Dancers sitting in circle, planning their piece
- Close-ups of hands, feet, movement details
- Empty studio before class (inviting, full of potential)
- Post-rehearsal moments (community, connection)

### What to Avoid

- Stock photos of "happy people dancing"
- Overly polished, photoshoot-style images
- Generic fitness/wellness imagery
- Posed, unnatural moments
- Images with no context or studio feeling

### Image Sizing & Placement

- Hero image: Full width, 400-600px height (mobile-responsive)
- Section images: Maximum 600px width, centered
- Use image filters sparingly — let real photography shine
- Alt text required on all images for accessibility

- Effect: Artistic, craft-focused
- Opacity: 10-20%

### Concrete (Subtle) (#3A3A3A base)
- Use on: Dark CTA sections
- Effect: Studio floor, grounded, modern
- Opacity: 5-10%

**Texture Implementation:**
- Use SVG filters or CSS patterns sparingly
- Never overpower readability with texture
- Mobile devices: Reduce texture opacity by half
- Always provide a solid fallback color

## 8. WRITING & VOICE

(This complements the visual identity)

### Copy Tone

- **Respectful of craft** — Dance is serious art, not casual hobby
- **Encouraging but authentic** — No toxic positivity or corporate-speak
- **Insider language** — Use dance terminology naturally ("choreography," "rehearsal," "performance")
- **Story-driven** — Lead with problem/solution, not features
- **Warm + professional** — Like talking to an experienced dance mentor

### Messaging Hierarchy

1. **Lead with story:** "We talked to 50+ dancers about what they need..."
2. **Validate the problem:** "The dance community is scattered..."
3. **Present the solution:** "Atelier brings it together..."
4. **Show the benefit:** "Find collaborators. Book spaces. Create together."
5. **Call to action:** "Be a founding member. Help shape Atelier."

### Examples of Brand Voice

**DON'T:**
- "Easy booking in 3 clicks!" (too transactional)
- "Disrupt the dance industry" (startup cliché)
- "Find your tribe" (too casual)
- "Join the movement!" (conflates metaphor with dance)

**DO:**
- "A creative home for dancers to collaborate and create"
- "Built by dancers, for dancers"
- "Where rehearsals become collaborations"
- "Stop wasting hours coordinating through DMs"
- "One platform. One home."

## 9. RESPONSIVE DESIGN RULES

### Mobile (< 768px)

- **Padding:** 20px sides instead of 40px
- **Font sizes:** Reduce H1 by 30% (56px → 36px)
- **Spacing:** Tighter vertical rhythm (40px sections become 24px)
- **Images:** Full width with no external margins
- **Navigation:** Single column, accessible hamburger menu

### Tablet (768px - 1024px)

- **Columns:** 2-column layouts
- **Container width:** ~768px
- **Font sizes:** Medium, between mobile and desktop
- **Spacing:** Medium, balanced

### Desktop (> 1024px)

- **Columns:** 2-3 column layouts
- **Container width:** 1100px max, centered
- **Font sizes:** Full sizes (56px H1, 18px body)
- **Spacing:** Generous (40px+ between sections)

### Mobile-First Development

1. Design mobile first
2. Add tablet breakpoints (@768px)
3. Add desktop breakpoints (@1024px)
4. Test on real devices (iPhone, Android, iPad)
